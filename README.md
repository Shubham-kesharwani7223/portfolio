Personal Portfolio Website

A modern Full Stack 3D Portfolio Website built using the MERN Stack with 
interactive UI, animations, authentication system, contact functionality, and responsive design.

🌐 Live Demo

🔗https://skportfolio-r477.onrender.com/

✨ Features

    🎨 Modern 3D Portfolio UI
    ⚛️ React Frontend
    🚀 Node.js + Express Backend
    🍃 MongoDB Database Integration
    🔐 Authentication System
    ☁️ Cloudinary Image Upload
    📩 Contact Form with Mailtrap SMTP
    📱 Fully Responsive Design
    🎭 Interactive Animations
    🧠 AI Inspired Portfolio Sections
    🛠️ MERN Stack Architecture
    
🛠️ Tech Stack

  Frontend
      React.js
      Redux Toolkit
      Material UI
      React Router DOM
      React Icons
      Three.js
  Backend
      Node.js
      Express.js
      MongoDB
      Mongoose
      JWT Authentication
  Other Tools & Services
      Cloudinary
      Mailtrap SMTP
      Render Deployment
      Git & GitHub
